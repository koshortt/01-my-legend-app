# 핀볼 레이스 (Pinball Race)

핀볼 레이스는 단순 랜덤 추첨 앱이 아니라, 결과가 만들어지는 과정을 함께 관전하며 즐기는 `관전형 랜덤 레이스 게임`이다.

사용자는 참가자를 등록하고, 자동 생성된 공들이 핀볼 맵을 통과하며 충돌, 추월, 역전, 결승 경쟁을 벌이는 과정을 지켜본다.

## 핵심 컨셉

룰렛보다 재미있는 관전형 랜덤 레이스

## 주요 기능

MVP 기준 주요 기능:

- 참가자 등록
- 참가자별 공 자동 생성
- WINNER / LOSER 모드
- 물리 기반 레이스
- 장애물 충돌
- 카메라 추적
- 실시간 순위 계산
- 결과 화면

추후 확장 후보:

- 랜덤 이벤트
- 맵 추가
- 공 꾸미기
- Auto Director Camera
- 생존전
- 토너먼트
- 팀 나누기
- 룰렛 모드

## 기술 스택

- Next.js
- React
- TypeScript
- Tailwind CSS

세부 버전은 `package.json`을 기준으로 한다.

## 실행 방법

### 의존성 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 다음 주소로 접속한다.

```text
http://localhost:3000
```

### 빌드

```bash
npm run build
```

### 프로덕션 실행

```bash
npm run start
```

### 린트

```bash
npm run lint
```
