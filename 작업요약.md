# 작업요약.md

## 현재 상태

Sprint 2.2 기준으로 물리 튜닝과 레이스 움직임 원인성 개선 작업을 완료했다.

## 구현 완료

- 참가자 등록
- 공 자동 생성
- WINNER / LOSER 모드
- 이름 표시
- 실시간 TOP 5 순위판
- NEW LEADER 표시
- 선두 추적 카메라
- 분기 구조
- 스피너, 해머, 이동 플랫폼, 점프 구간
- 가이드 바, C자 함정, S자 통로
- 정체 방지 시스템
- 바람 구간

## Sprint 2.2 작업 내용

- 공이 제멋대로 좌우로 움직이는 느낌을 줄이기 위해 매 프레임 무작위 수평 힘을 제거했다.
- 바람 구간을 추가하여 좌우 이동의 시각적 원인을 만들었다.
- 중력과 최대 낙하 속도를 조정해 경기 시간이 너무 느려지지 않도록 했다.
- 수평 최대 속도를 낮춰 옆으로 날아가는 느낌을 줄였다.
- 가이드 바, 스피너, 해머 충돌 보정값을 낮춰 떨림과 과한 튕김을 줄였다.
- 자동 탈출 시스템은 유지하되 랜덤 밀기 대신 중앙 방향 밀기로 변경했다.

## 주요 값

- gravity: `0.066`
- friction: `0.982`
- damping: `0.994`
- restitution: `0.52`
- maxFallSpeed: `5.1`
- maxHorizontalSpeed: `1.65`
- physics step: `1000 / 60`

## 검증 결과

- `npm run lint` 통과
- `npm run build` 통과
- 로컬 서버 `http://localhost:3000` 응답 확인
- 화면 버전 `Sprint 2.2` 반영 확인
- 물리 시뮬레이션 기준 평균 첫 도착 약 `27.4초`
- 물리 시뮬레이션 기준 평균 전체 종료 약 `34.9초`
