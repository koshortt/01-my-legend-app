# 다음_작업.md

## 바로 다음 작업

Sprint 2.3 후보: 브라우저 실제 플레이 기준 레이스 감성 미세 조정

## 작업 목표

Sprint 2.2에서 조정한 공 떨림 제거, 바람 구간, 낙하 속도 변경이 실제 브라우저 관전에서 자연스럽게 보이는지 확인한다.

## 작업 범위

- 실제 브라우저에서 2~3회 레이스 확인
- 경기 시간이 25~35초 범위에 드는지 확인
- 바람 구간 밖에서 원인 없는 좌우 움직임이 줄었는지 확인
- 공이 장애물 위에서 덜덜 떨거나 오래 정체되는지 확인
- 필요 시 물리 상수만 소폭 조정

## 구현 금지

- Auto Director Camera 확장
- BIG MOVE
- 이벤트 로그 확장
- 리플레이
- 통계 시스템
- 신규 맵 추가
- 캐릭터 시스템

## 완료 기준

- 경기 시간이 25~35초 수준으로 유지된다.
- 공의 기본 흐름이 아래쪽 중심으로 보인다.
- 좌우 이동의 원인이 장애물, 분기, 바람 구간으로 인지된다.
- 공 떨림과 물리 버그처럼 보이는 움직임이 줄어든다.
