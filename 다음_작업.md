# 다음_작업.md

## 바로 다음 작업

Classic Blueprint v1 실제 브라우저 검증

## 작업 목표

Classic Map Blueprint v1 기준으로 재구축한 맵이 실제 플레이에서 컨셉 이미지처럼 레이스 트랙으로 보이는지 확인한다.

## 작업 범위

- 실제 브라우저에서 3판 이상 레이스 실행
- 첫 화면에서 A / B / C 루트가 즉시 보이는지 확인
- MERGE에서 참가자들이 실제로 만나는지 확인
- S COURSE에서 좌우 이동이 충분히 발생하는지 확인
- FAST / SAFE / CHAOS 루트 차이가 체감되는지 확인
- BOTTLENECK에서 TOP5가 한 화면에 모이는지 확인
- SCRAMBLE에서 결승 직전 순위 변동이 발생하는지 확인
- 필요 시 Blueprint 범위 안에서만 좌표 미세 조정

## 구현 금지

- Blueprint 외 임의 루트 추가
- Blueprint 외 임의 장애물 추가
- 신규 맵 추가
- 리플레이 기능
- 통계 기능
- 승률 기능
- 커스터마이징 화면
- 대규모 구조 리팩토링

## 완료 기준

- 화면만 봐도 "레이스 트랙"으로 보인다.
- A / B / C가 명확히 구분된다.
- FAST / SAFE / CHAOS가 명확히 구분된다.
- 참가자가 장애물보다 먼저 보인다.
- 공이 장애물에 3초 이상 갇히지 않는다.
- 결승에서 TOP5가 한 화면에 모인다.
